# Website Starter Pack - Simply App

- Webpack 
    - Babel, 
    - CSS, 
    - JS (ES2015, ES2016 & ES2017), Async/Await
    - FontAwesome 5
    - Fonts
    - Webpack serve
- HTML template

## Usage

### Installation

Install the dependencies

```sh
$ yarn install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ yarn start
```

### Build
Compile and build

```sh
$ yarn build
```
